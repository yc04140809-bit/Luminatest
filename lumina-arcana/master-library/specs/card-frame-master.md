## 目的(ユーザー制定・原文)

カードイラストではなく、**「Lumina Arcana と一目で分かるブランドフレーム」**を作ること。

## 守ること(ユーザー制定・原文)

- 世界観を最優先
- 植物モチーフ
- アールヌーボー調
- 左右対称
- 余白を大切にする
- 高級感
- 印刷対応
- アプリ表示対応

## カード構成(ユーザー制定・原文)

| エリア | 位置 | 内容 |
|---|---|---|
| ① シンボルエリア | 上部 | 月・星 |
| ② イラストエリア | 中央 | カード全体の約65% |
| ③ テキストエリア | 下部 | カード名 / メッセージ / キーワード |

## 禁止事項(ユーザー制定・原文)

- 派手な装飾
- 可読性を下げる装飾
- イラストより目立つフレーム
- 世界観を壊す色

## マスター画像の登録内容(v001 デザインシートより)

マスター画像は以下を1枚に含むデザインシートである。

### カード裏面(Card Back Design Master v1.0)

- コンセプト: **時のゆりかご(Time Cradle)** — 「すべての心が帰ってこられる場所。生命が静かに育ち、月の光と希望に包まれる世界。」
- 深い紺青の地に、金の植物装飾(アールヌーボー調・左右対称)、中央に三日月と一番星、下部に水面

### カード表面(Card Front Frame Master v1.0)

- フレームの特徴: 統一された世界観 / アールヌーボー調の植物装飾 / 上部に月と星の意匠 / 下部にメッセージエリア / 余白を大切にしたデザイン
- 構成エリア: シンボルエリア(月と星)→ イラストエリア → メッセージエリア(今日のあなたへ)→ キーワードエリア(深く読み解く)→ 装飾モチーフエリア(自然の息吹)

### 正式カラーパレット(Color Palette)

| 色名 | 日本語 | 用途イメージ |
|---|---|---|
| Deep Midnight Blue | 深い紺青 | ベース(裏面・夜) |
| Moon Ivory | 月のアイボリー | 表面テキストエリア・余白 |
| Antique Gold | アンティークゴールド | フレーム装飾・箔 |
| Soft Emerald | 深緑 | 植物モチーフ |
| Lavender Silver | ラベンダーシルバー | 副装飾 |
| Warm Light | 温かな光 | 光の粒・アクセント |

※ 各色の正確なカラーコードは【未設定】(マスター画像が正。実装時に採寸して本書へ登録する)

### 質感・仕上げ(Texture / Finish)— 印刷対応

エンボス加工(深き彫り)/ 金箔押し(ゴールド箔)/ マット紙質感(高級厚紙)/ 繊細な刻印(線の彫り)

### 季節バリエーション(Seasonal Variations)

| 季節 | テーマ |
|---|---|
| 春 Spring | 若葉が芽吹く季節 |
| 夏 Summer | 深い緑と月明かり |
| 秋 Autumn | 紅葉とアンティークゴールド |
| 冬 Winter | 雪化粧と静謐の銀 |

### 隠された象徴(Hidden Symbolism)

| 意匠 | 象徴 |
|---|---|
| 三日月 | 受容と直感 |
| 一番星 | 希望と導き |
| ゆりかご | 時のゆりかご |
| 枝葉 | 生命の成長 |
| 円環 | 循環と永遠のつながり |
| 水面 | 心の静けさと帰還 |

※ 「ゆりかご」のモチーフは、正式名称では聖域「生命の揺り籠」を指す(世界=時のゆりかご/聖域=生命の揺り籠。2026-07-18 統一)

## 管理画面からの変更設計(ユーザー制定)

管理画面から以下を変更できる構造とする(実装時のデータ設計指針)。

| 変更項目 | 変数(実装時の想定キー) | 備考 |
|---|---|---|
| フレーム色 | `frame.color` | 正式パレットの範囲内 |
| 装飾 | `frame.ornament` | 禁止事項に反しないもの |
| 季節テーマ | `frame.season` | spring / summer / autumn / winter(+イベント) |
| 箔色 | `frame.foil` | 印刷・表示両対応 |
| 背景 | `frame.background` | 世界設定([10章](../10_app-specifications.md))と連動 |

- フレーム画像はイラスト・テキストと分離したレイヤー/アセットとして管理し、差し替えてもカードデータ(Card ID・文言)に影響しない構造とする
