## 1. トップ画面

### 構成

| 要素 | 内容 | 状態 |
|---|---|---|
| 背景 | 世界そのもの(時のゆりかごの風景)。季節・世界設定により変更可能 | 【正式設定】 |
| 環境アニメーション | 葉が揺れる / 蛍 / 光の粒 / 月明かり / 水面 のみ(最低限) | 【正式設定】 |
| 「今日のカード」ボタン | Card Ceremony の起点。押下でボタン沈み込み→水の波紋 | 【正式設定】 |
| その他のUI要素(ジャーナル導線・設定・管理ページ導線 等) | 配置・数・表示方法 | 【未設定】 |
| BGM | オープニングから自然に引き継ぐ | 【正式設定】 |

### ルール

- UIは固定。背景のみ季節変更可能
- ユーザーを急がせる要素(カウントダウン・バッジ・通知強調)を置かない
- 画面の大部分は「世界」に譲る。UIは世界の上に静かに浮かぶ

## 2. カード画面①(カードとの出会い)

### 構成

| 要素 | 内容 |
|---|---|
| カード | 大きく表示。この画面の主役 |
| 見出し | 「今日のあなたへ」 |
| 短いメッセージ | カードの基本メッセージ(messages.base) |
| ボタン | 「メッセージを読み解く」(画面②へ) |

### ルール

- 情報量は最小。余白を最大
- カードを眺める時間を作る(自動遷移・時間制限をつけない)
- スクロールなしで完結する構成を基本とする

## 3. カード画面②(カードを読み解く)

### 構成

| 要素 | 内容 |
|---|---|
| 見出し | 「今日のあなたへ」 |
| 深く読み解く | キーワード(keywords)/ 意味(story + messages.base)/ アドバイス(messages.action ほか)/ 注意点(messages.caution) |
| 導線 | ジャーナルへ / 過去を見る |

### ルール

- 画面①より情報量は多くてよいが、読みやすさを最優先
- 長文は見出し・余白で区切り、圧迫感を作らない
- データ項目との対応は [10_アプリ仕様](../10_app-specifications.md) の対応表が正

## 5. UIルール(全画面共通)

1. **引き算の美学**: 情報を詰め込まない。余白を作る。安心を作る
2. **カードが主役**: UIがカードや世界より目立ってはならない
3. **急がせない**: 時間制限・連打誘導・強制アニメーションを置かない(オープニングはスキップ可能)
4. **画面遷移**: 波紋 / フェード / 光 / 風 のみ。派手なトランジション禁止
5. **演出禁止事項**: ガチャ演出・ゲーム的リザルト・ランキング・恐怖や焦りを与える表現
6. **世界の呼吸**: 静止画面でも環境アニメーションで「世界が生きている」ことを感じさせる
7. **文言**: すべて [08_ライティングガイドライン](../08_writing-guidelines.md) に従う(断定しない・急かさない・裁かない)
8. **対応環境**: スマホ最優先。PCでも崩れないレスポンシブ構成
9. フォント・ボタン形状・角丸・余白の具体値: 【未設定】(実装着手時に本書へ登録)

## 6. 配色

方針(【正式設定】): 淡く透明感のある配色。彩度の高すぎる原色・攻撃的な赤黒は使わない([07_画像生成ガイドライン](../07_image-generation-guidelines.md) と共通)。

**ブランドの正式パレットは [Card Frame Master v1.0](./card-frame-master.md) の6色が正**(Deep Midnight Blue / Moon Ivory / Antique Gold / Soft Emerald / Lavender Silver / Warm Light。2026-07-18 ユーザー採用)。
アプリUIの配色もこの6色を基準に組む。

下記のUI用ドラフト(【仮設定】)は正式パレットとの整合待ち。実装着手時に正式パレットの実測値で置き換える(【要確認】):

| 役割 | 色名(仮) | 参考値(仮) | 由来 |
|---|---|---|---|
| ベース(夜) | 宵藍(よいあい) | #1B2440 | 夜の世界・月明かりの空 |
| ベース(明) | 月白(げっぱく) | #F5F2E9 | 月あかり・余白 |
| アクセント(光) | 灯金(ともしがね) | #E8C87A | 生命の揺り籠に灯る光・光の粒 |
| サブ(水) | 水鏡(みずかがみ) | #7FAFB8 | 水面・波紋 |
| サブ(緑) | 若葉 | #9DB89A | 植物・生命 |
| テキスト | 墨鼠(すみねず) | #3A3A40 | 明背景上の本文 |

- 警告色・エラー色にも強い赤は使わず、やわらかい表現にする(具体値【未設定】)
- 季節・世界設定(春夏秋冬・雨・雪・満月・新月・イベント)ごとの派生パレットは【未設定】(世界設定機能の実装時に登録)
