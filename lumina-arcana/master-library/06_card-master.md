# 06 カードマスター

- **ステータス**: 運用中(78枚の進捗管理台帳)
- **最終更新**: 2026-07-16

全78枚のカードの**唯一の正式な一覧表**です。
Card ID の発番・タイトル・制作ステータスはこの表で管理します。
カード本体のデータは [Card Database](../card-database/) の個別ファイルが正です。

## ステータス定義

| ステータス | 意味 |
|---|---|
| `planned` | 発番のみ。内容未着手 |
| `draft` | 執筆・制作中 |
| `review` | レビュー待ち |
| `released` | 正式版。アプリ・SNS等で使用可 |
| `revising` | 公開後の改訂作業中 |
