## 1. 存在理由

Lumina Arcana は「オラクルカードアプリ」を作るためのプロジェクトではない。

**世界中の人が安心して帰ってこられる場所『時のゆりかご』を提供するために存在する。**

## 2. 使命

- 『時のゆりかご』を体験として届ける
- カードは未来を当てるものではなく、**今日という一日に寄り添う存在**として差し出す
- 世界そのものが「おかえり。」と迎えてくれる体験を提供する

## 3. 価値観

### 理念の核(すべての判断基準)

安心 / 癒し / 優しさ / 希望 / 心を整理する時間 / 依存させない / 恐怖で煽らない / 未来を断定しない / 自分で答えを見つけられる設計

### 体験として最優先する価値

安心 / 温もり / 余白 / 静寂 / 自然 / 生命 / 対話 / 自己受容

| 価値 | 具体的な指針 |
|---|---|
| 安心 | 否定しない。裁かない。急かさない |
| 癒し・温もり | 攻撃的な色・音・言葉を使わない |
| 希望 | 悪い結果を断定しない。必ず前向きな選択肢を残す |
| 余白・静寂 | 情報過多にしない。余白を設計する |
| 心を整理する時間 | 自分の気持ちと静かに向き合う時間を提供する |

## 4. 守ること

1. ブランド理念を利便性より優先する
2. 世界そのものがユーザーを迎え入れることを最優先する
3. 利用者が自分で答えを見つけられる設計にする
4. Master Library(世界の正典)との整合を保つ
5. 静けさ・余白・引き算の美学を保つ

## 5. 守らないこと(禁則)

1. 不安や恐怖を煽って行動させること(「〜しないと不幸になる」等)
2. 未来の断定(「必ず〜になります」)
3. 依存を促す設計・過度な課金誘導・射幸性(ガチャ演出)
4. 派手なゲーム演出・利用者を急がせる演出
5. 他者・他ブランド・他の占術への否定や攻撃
6. 世界観を壊す安易なコラボ・流行への迎合
7. Lumina Arcana を「占いアプリ」「ゲーム」「SNS」「AIチャット」にすること

## 6. ブランド哲学

> Lumina Arcana は
> 占いアプリではない。ゲームでもない。SNSでもない。AIチャットでもない。
>
> ここは『時のゆりかご』。
> 世界中の人が安心して帰ってこられる場所。
>
> カードは未来を当てるものではなく、
> 今日という一日に寄り添う存在。
>
> 世界そのものが「おかえり。」と迎えてくれる体験を提供する。
>
> これが Lumina Arcana のブランド哲学である。

## ブランドの中心 = 世界『時のゆりかご』

Lumina Arcana の世界そのものが「時のゆりかご」であり、その中心には聖域「生命の揺り籠」がある(世界=時のゆりかご/聖域=生命の揺り籠。2026-07-18 統一)。
プロダクトのすべては「利用者が時のゆりかごに帰ってくる体験」として設計する。

→ 詳細: [02_時のゆりかご](./02_time-cradle.md) / [World Bible](../world-bible/)
