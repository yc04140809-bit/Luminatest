# 12 収益モデル

- **ステータス**: 構想段階
- **最終更新**: 2026-07-16

収益はブランドを長く続けるための手段であり、目的ではありません。
**「依存させて課金させる」型の収益は採用しない**ことを最初に明記します。

## 収益の柱(構想)

| 柱 | 内容 | フェーズ | 備考 |
|---|---|---|---|
| 紙カード販売 | 物理デッキ・特装版 | Phase 3 | 単発購入型。IPの旗艦商品 |
| 電子書籍 | 世界観ブック・解説書 | Phase 3〜4 | ストック収益 |
| アプリ | 基本無料+買い切り/穏やかなサブスク | Phase 3〜4 | 下記の課金原則に従う |
| グッズ | キャラクター・世界観グッズ | Phase 4〜5 | 小ロットから |
| コミュニティ | 有料会員(検討) | Phase 4〜5 | 無料でも安心して居られる場を維持 |

## 課金の原則

1. 無料ユーザーの体験だけでも「安心・癒し」が完結すること
2. ガチャ的な射幸性を持つ課金は行わない
3. 「課金しないと不安が解消されない」構造を作らない(理念違反)
4. 価格は「毎日の小さな安心」に見合う、無理のない水準にする

## 記録ルール

- 実際に収益化を開始したら、この章に「開始日・形態・価格」を記録する
- 撤退・変更もすべて記録する(意思決定の履歴を残す)
