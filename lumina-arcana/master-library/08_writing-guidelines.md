# 08 ライティングガイドライン

- **ステータス**: 正式版 v1.0
- **最終更新**: 2026-07-16

カード文言・SNS・アプリ内テキストなど、Lumina Arcana が発するすべての言葉のルールです。
**誰が(どのAIが)書いても同じ声になる**ことを目的とします。

## ブランドの声(Voice)

- 静かで、あたたかく、そっと隣に座るような声
- 上から教えるのではなく、隣で一緒に眺める
- 押し付けない。急かさない。決めつけない

## 文体ルール

| 項目 | ルール | 例 |
|---|---|---|
| 語りかけ | やさしい「です・ます」調 | 「〜かもしれません」「〜してみませんか」 |
| 断定 | 未来を断定しない | ×「必ず成功します」→ ○「うまくいく流れが近づいています」 |
| 否定 | 利用者を否定しない | ×「あなたの考えは間違いです」 |
| 命令 | 命令形を避け、提案形にする | ×「〜しなさい」→ ○「〜してみるのもいいかもしれません」 |
| 脅し | 恐怖で動かさない | ×「このままだと不幸になります」 |
| 絵文字 | SNSでは控えめに可。カード本文では使わない | |

## 項目別の書き方(カード)

| 項目 | 目安 | 書き方 |
|---|---|---|
| ストーリー | 100〜200字 | カードの情景を物語として描く。世界観と接続する |
| 基本メッセージ | 50〜100字 | カードが今日そっと手渡す言葉 |
| 恋愛/仕事/金運/人間関係 | 各50〜80字 | 前向きな選択肢を必ず1つ残す |
| 今日の行動 | 1文 | 5分以内にできる小さな行動 |
| 注意点 | 1〜2文 | 「気をつけて」ではなく「無理しないで」の方向で書く |

## NGワード・要注意表現

- 「絶対」「必ず」「二度と」「手遅れ」「最悪」「呪い」「祟り」
- 病気・死・法律・投資について断定的な助言をしない(専門家の領域に踏み込まない)
- 他の占い・宗教・思想への言及や比較

## 表記統一

- 固有名詞は [03_用語集](./03_glossary.md) の正式表記に従う
- 数字は半角、感嘆符・疑問符は全角(!?)を基本とする
- 三点リーダーは「……」(2つ重ね)
