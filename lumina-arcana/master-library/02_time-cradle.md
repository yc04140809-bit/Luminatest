## 世界「時のゆりかご」

ブランド理念「安心・癒し・優しさ・希望・心を整理する時間」を体現する世界。

- コンセプト: すべての心が帰ってこられる場所。生命が静かに育ち、月の光と希望に包まれる世界
- 基本要素(昼夜・季節・自然・存在)と「やらないこと」: [01_世界観](./01_worldview.md) が正

## 存在理由【正式設定】(v1.0・2026-07-18)

公式文(設定資料・アプリ・書籍で共通して使用する):

> 時のゆりかごは、人を変えるためにあるのではない。
> 未来を当てるためにあるのでもない。
> **人々が安心して帰ってこられる場所**として存在する。
>
> この世界は温もりと生命に満ち、訪れる人を選ばず、拒まず、
> ありのままの心をそのまま受け入れる(自己受容)。
> そして、今日という一日に小さな希望の灯りを添えて、
> 「いってらっしゃい。」と日常へ送り出す。

- 核となるキーワード: 安心 / 温もり / 希望 / 自己受容 / 生命 / 帰ってこられる場所 / 今日という一日に小さな灯りを添える
- カードは、この世界から日常へ持ち帰る「小さな希望の灯り」として生まれる

## 空間設定【正式設定】(v1.0・2026-07-18)

世界は、中心「生命の揺り籠」から外へ向かって自然に広がる(円環=循環と永遠のつながり)。

| 位置 | 場所 | 内容 |
|---|---|---|
| 中心 | **世界樹** | 世界の中心に立つ大樹。その枝葉が絡み合って生命の揺り籠を形づくり、抱いている |
| 中心 | **生命の揺り籠** | 世界樹の枝葉に抱かれた揺り籠。カードが静かに生まれる聖域 |
| 中心のまわり | **湖** | 世界樹と生命の揺り籠を抱く静かな湖。水面は月と星の光を映す |
| 湖のほとり | **丘** | やわらかな草の丘。月と星空を眺め、心を休められる場所 |
| 外側 | **森** | 湖と丘を囲む深い森。蛍が灯り、風が葉を揺らす |
| つなぐもの | **小道** | 丘・森・湖のほとりを静かに結ぶ道。急がなくていい道 |
| 天 | **月と星空** | 世界を照らす三日月(受容と直感)と一番星(希望と導き) |

- **拡張の仕組み**: 新しい場所(建物・土地)は「小道の先」に増やしていく。中心構造(世界樹・生命の揺り籠・湖)は変更しない
- 世界の広さ・果ては定義しない(必要になるまで決めない。複雑化を避ける)
- 地図は後から追加する。文章ベースの空間設計が正: [world-bible/world-map.md](../world-bible/world-map.md) で管理

## 時間の流れ方【正式設定】(v1.0・2026-07-18)

- 時のゆりかごにも、現実と同じように時間は流れる。朝 → 昼 → 夕 → 夜が巡り、春 → 夏 → 秋 → 冬が巡る
- ただし、この世界の時間は**穏やかで、呼吸するように**流れる。誰かを急かすためではなく、心を整えるために流れる時間である(ユーザーを「急かされる世界」に置かない)
- 天候がある: 晴れ・雨・雪など。雨も雪も世界の一部として静かに描く(災害・恐怖としては描かない)
- 月は満ち欠けする(満月・新月)。星空は夜ごとに世界を見守る
- 巡りは終わりではなく、循環と再生のしるし。過去は水面に映る景色のように、静かに眺められるもの

### 演出への反映構造(アプリ・カードとの接続)

世界の時間・季節・天候は、そのまま演出の変数になる(→ [10_アプリ仕様](./10_app-specifications.md) の世界設定機能と対応)。

| 世界の要素 | 反映先 |
|---|---|
| 昼夜(朝・昼・夕・夜) | 背景の光・BGM・波紋色 |
| 四季(春・夏・秋・冬) | 季節背景・カードフレームの季節バリエーション・BGM |
| 天候(雨・雪 等) | 背景・環境音・波紋色 |
| 月(満月・新月) | 夜の背景・光の演出 |
| 蛍・光の粒 | 環境アニメーション・Card Ceremony の光 |

## 世界の内的ルール【正式設定】(v1.0・2026-07-18)

戦闘システムのようなゲームルールではなく、世界そのものの約束事。

1. **世界は常に呼吸している。** 葉が揺れ、水面がゆらぎ、光の粒が漂う。世界が完全に静止することはない
2. **自然は生命を育む。** 森・湖・世界樹のもとで、生命は静かに育つ
3. **この世界は誰も拒まない。誰も裁かない。誰も急かさない。** 訪れるのに資格はいらない
4. **争いではなく、対話を大切にする。** この世界に戦いはない。心は言葉と静けさの中で整えられる
5. **カードは引くものではなく、訪れるもの。** カードは生命の揺り籠で生まれ、訪れた人のもとへ届く
6. **夜も雨も、世界の一部。** 闇や揺らぎも排除されず、静かに受け止められる
7. **世界は訪れた人を引き留めない。** 帰り道はいつもひらかれていて、「いってらっしゃい。」と日常へ送り出す(依存させない設計の世界観的な根拠)
8. **世界のすべては静かに循環する。** 昼と夜、季節、訪れと帰り。だからいつでも、また帰ってこられる

## 聖域「生命の揺り籠」

> 【正式設定】カードが静かに生まれ、光が集まり、生命と希望が育まれる神聖な場所。
> Card Ceremony でカードが訪れる起点となる。

### 設定項目

| 項目 | 内容 |
|---|---|
| 場所 | 世界「時のゆりかご」の中心。世界樹の枝葉に抱かれ、湖に浮かぶ【正式設定 v1.0】 |
| 役割 | カードの誕生地。Card Ceremony の起点【正式設定】 |
| 見た目 | 世界樹の枝葉が絡み合って形づくられた揺り籠。夜には光の粒が集まり、灯りがともる(オープニング演出・カード裏面モチーフと共通)【正式設定 v1.0】 |
| 世界樹との関係 | 世界樹の枝葉が生命の揺り籠を形づくり、抱いている【正式設定 v1.0】 |
| たどり着き方 | 【未設定】(利用者視点では「アプリを開く=帰ってくる」で足りるため、必要になるまで定義しない) |
| 管理者/住人 | 【未設定】(ルミナとの関係は Phase 1-C で確定) |

## 感情設計【正式設定】(ユーザー決定 2026-07-18)

Lumina Arcana が目指す体験の核:

> **「小さな希望を持って日常へ戻れること」**

- 利用者がアプリを閉じるとき、少しだけ心が軽くなり、「今日はもう少し頑張ってみよう。」「また帰ってこよう。」と自然に思える体験を、ブランド全体の設計思想とする
- ブランドは人生を変えることを約束しない。未来を当てることも目的ではない
- 存在意義: **今日という一日に、小さな希望の灯りを添えること**
- 体験の入口と出口: 開始時「**おかえり。**」/ 終了時「**いってらっしゃい。**」(ブランド全体に反映する)

## 体験としての「時のゆりかご」

プロダクト設計では、利用者の体験を「世界・時のゆりかごに帰ってくること」として統一する。

- アプリを開く = 時のゆりかごに帰ってくる(「おかえり。」)
- カードが訪れる = 生命の揺り籠で生まれたカードが届く(Card Ceremony)
- アプリを閉じる = 小さな希望を持って日常へ戻る(「いってらっしゃい。」)
- SNSの投稿 = 時のゆりかごから届く小さな便り

## 禁則

- 時のゆりかご・生命の揺り籠を「試練の場」「裁きの場」として描かない
- 恐怖・焦り・締め切りの感覚を持ち込まない
