## 世界「時のゆりかご」

ブランド理念「安心・癒し・優しさ・希望・心を整理する時間」を体現する世界。

- コンセプト: すべての心が帰ってこられる場所。生命が静かに育ち、月の光と希望に包まれる世界
- 基本要素(昼夜・季節・自然・存在)と「やらないこと」: [01_世界観](./01_worldview.md) が正

## 存在理由【正式設定】(v1.0・2026-07-18)

時のゆりかごは、現実のどこかにある場所ではない。
**疲れた心が「帰ってくる」ためにひらかれた世界**である。

- 誰の心の中にもある「安心して戻れる場所」が、ひとつの世界として形になったもの
- この世界は、訪れる人を選ばず、拒まず、いつでも「おかえり。」と迎えるために在る
- カードは、この世界から日常へ持ち帰る「小さな希望の灯り」として生まれる

## 空間設定【正式設定】(v1.0・2026-07-18)

世界は、中心から外へ向かって静かな円環をなしている(円環=循環と永遠のつながり)。

| 層 | 場所 | 内容 |
|---|---|---|
| 中心 | **生命の揺り籠** | 世界樹の枝葉が絡み合って形づくられた揺り籠。カードが静かに生まれる聖域 |
| 中心 | **世界樹** | 世界の中心に立つ大樹。その枝葉が生命の揺り籠を抱いている |
| 内側 | **湖** | 世界樹と生命の揺り籠を抱く静かな湖。水面は心の静けさと帰還を映す |
| 外側 | **森** | 湖を囲む深い森。蛍が灯り、風が葉を揺らす |
| 天 | **月と星** | 世界を照らす三日月(受容と直感)と一番星(希望と導き) |

- 世界の広さ・果て・その他の土地は【未設定】(必要になるまで定義しない。複雑化を避ける)
- 地図上の表現は [world-bible/world-map.md](../world-bible/world-map.md) で管理

## 時間の流れ方【正式設定】(v1.0・2026-07-18)

- 時のゆりかごの時間は、現実より少しだけゆっくり流れる。誰かを急かすためではなく、**心を整えるために流れる時間**である
- 朝 → 昼 → 夕 → 夜が巡り、春 → 夏 → 秋 → 冬が巡る。巡りは終わりではなく、循環と再生のしるし
- 過去は責めるためのものではなく、水面に映る景色のように、静かに眺められるもの
- 時間は敵ではなく、心を癒す味方として描く(世界観の原則と共通)

## 世界の内的ルール【正式設定】(v1.0・2026-07-18)

1. **この世界は誰も拒まない。** 訪れるのに資格はいらない
2. **この世界では誰も裁かれない。** 急かされることもない
3. **夜も雨も、世界の一部。** 闇や揺らぎも排除されず、静かに受け止められる
4. **カードは生命の揺り籠で生まれ、訪れた人のもとへ「訪れる」。** 引き当てるものではなく、届くもの
5. **世界は訪れた人を引き留めない。** 帰り道はいつもひらかれていて、「いってらっしゃい。」と日常へ送り出す(依存させない設計の世界観的な根拠)
6. **世界のすべては静かに循環する。** 昼と夜、季節、訪れと帰り。だからいつでも、また帰ってこられる

## 聖域「生命の揺り籠」

> 【正式設定】カードが静かに生まれ、光が集まり、生命と希望が育まれる神聖な場所。
> Card Ceremony でカードが訪れる起点となる。

| 項目 | 内容 |

| 役割 | カードの誕生地。Card Ceremony の起点【正式設定】 |
| 見た目 | 世界樹の枝葉が絡み合って形づくられた揺り籠。夜には光の粒が集まり、灯りがともる(オープニング演出・カード裏面モチーフと共通)【正式設定 v1.0】 |

## 感情設計【正式設定】(ユーザー決定 2026-07-18)

Lumina Arcana が目指す体験の核:

> **「小さな希望を持って日常へ戻れること」**

- 利用者がアプリを閉じるとき、少しだけ心が軽くなり、「今日はもう少し頑張ってみよう。」「また帰ってこよう。」と自然に思える体験を、ブランド全体の設計思想とする
- ブランドは人生を変えることを約束しない。未来を当てることも目的ではない
- 存在意義: **今日という一日に、小さな希望の灯りを添えること**
- 体験の入口と出口: 開始時「**おかえり。**」/ 終了時「**いってらっしゃい。**」(ブランド全体に反映する)

## 体験としての「時のゆりかご」

プロダクト設計では、利用者の体験を「世界・時のゆりかごに帰ってくること」として統一する。

- アプリを開く = 時のゆりかごに帰ってくる(「おかえり。」)
- カードが訪れる = 生命の揺り籠で生まれたカードが届く(Card Ceremony)
- アプリを閉じる = 小さな希望を持って日常へ戻る(「いってらっしゃい。」)
- SNSの投稿 = 時のゆりかごから届く小さな便り

## 禁則

- 時のゆりかご・生命の揺り籠を「試練の場」「裁きの場」として描かない
- 恐怖・焦り・締め切りの感覚を持ち込まない
