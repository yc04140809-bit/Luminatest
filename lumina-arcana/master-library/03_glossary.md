# 03 用語集

- **ステータス**: 運用中(用語が増えるたびに追記)
- **最終更新**: 2026-07-16

Lumina Arcana の公式用語の一覧です。
**表記ゆれを防ぐため、世界観・カード・SNS・アプリのすべてでこの表記を使用します。**
長い解説が必要な用語は [world-bible/terms/](../world-bible/terms/) に個別ファイルを作り、ここからリンクします。

## 記載ルール

- 五十音順に並べる
- 「正式表記」「英語表記」「ひとこと定義」「詳細リンク」を必ず埋める
- 廃止した用語は削除せず「廃止」欄に移動する(過去資料の解読のため)
