# 01 世界観

- **ステータス**: 基本要素【正式設定】(2026-07-18)/ 詳細は設定中
- **最終更新**: 2026-07-18

この章は Lumina Arcana 世界の「公式サマリー」です。
詳細設定は [World Bible](../world-bible/) に置き、この章では全体像と原則のみを定義します。

## 世界の全体像

- 世界の名称: **時のゆりかご**【正式設定】(ユーザー決定 2026-07-18)
- 世界のトーン: やわらかい光、静けさ、時間がゆっくり流れる感覚
- コンセプト: すべての心が帰ってこられる場所。生命が静かに育ち、月の光と希望に包まれる世界(Card Frame Master v1.0 より)
- 詳細: [02_時のゆりかご](./02_cradle-of-time.md)

## 世界の基本要素【正式設定】(ユーザー決定 2026-07-18)

| 区分 | 要素 |
|---|---|
| 昼夜 | 朝 / 昼 / 夕 / 夜 |
| 季節 | 春 / 夏 / 秋 / 冬 |
| 自然 | 森 / 湖 / 月 / 蛍 / 風 / 星 |
| 存在 | ルミナ / カード / 揺り籠 / 世界樹 |

- カードは「世界の中に実在する存在」として扱う(単なる道具ではない)
- 「揺り籠」「世界樹」の詳細(姿・場所・関係)は【未設定】
- 昼夜・季節はアプリの世界設定(季節テーマ・背景変更)・カードフレームの季節バリエーションと連動する

## やらないこと(世界に持ち込まないもの)【正式設定】(ユーザー決定 2026-07-18)

機械的 / SF / ネオン / サイバー / ホラー / 暴力

## 世界観の原則(壊してはいけないもの)

## 世界観の原則(壊してはいけないもの)

1. この世界は訪れる人を**拒まない**。誰にとっても安全な場所である
2. 時間は敵ではなく、心を癒す味方として描かれる
3. 闇・混沌も「排除すべき悪」ではなく、受け止め方を学ぶ対象として描く(→ ケイオスちゃんの役割)
4. 現実世界の特定の宗教・政治・実在人物と直接結びつけない

## カード・キャラクターとの関係

- 78枚のカードは、この世界の風景・存在・出来事を切り取った「窓」である
- キャラクター(ルミナ達)は、利用者をこの世界へ案内するガイドである
- 世界観と矛盾するカード・キャラ設定は作らない(矛盾を見つけたら本章を正とする)
