# 11 ブランド展開

- **ステータス**: 構想段階
- **最終更新**: 2026-07-16

Lumina Arcana を長期IPとして育てるための展開領域の全体図です。
個別の実行計画は時期が来たら子ドキュメントに切り出します。

## 展開領域マップ

| 領域 | 内容 | 依存するもの | フェーズ |
|---|---|---|---|
| SNS | 世界観の発信・ファンとの接点 | カード数枚+画像スタイル確定 | Phase 2 |
| アプリ | 今日の一枚・カード図鑑 | 78枚のデータ完成 | Phase 3 |
| 紙カード | 物理オラクルカードの販売 | 全画像の印刷解像度マスター | Phase 3 |
| 電子書籍 | 世界観ブック・カード解説書 | Master Library の内容充実 | Phase 3〜4 |
| コミュニティ | 安心して集まれる場 | ファン層の形成 | Phase 4 |
| キャラクターIP | グッズ・物語コンテンツ | Character Bible の確立 | Phase 4〜5 |

## 展開の判断基準

新しい展開を始める前に、必ず以下を確認する。

1. ブランド理念「安心・癒し・希望・心を整理する時間」を強化するか?
2. 世界観(時のゆりかご)と矛盾しないか?
3. スマホ中心の体制で運営を続けられるか?(持続可能性)
4. 既存の展開の品質を下げないか?(手を広げすぎない)

4つすべて YES でなければ着手しない。

## 各展開の設計メモ

### 紙カード販売
- 印刷仕様(サイズ・紙質・箱)は未定。画像は最初から印刷解像度で保存しておく(→ [07_画像生成ガイドライン](./07_image-generation-guidelines.md))
- 小ロット印刷サービスの比較検討は Phase 3 開始時に実施

### 電子書籍
- Master Library の「世界観」「カード解説」がそのまま原稿の母体になる構造にしてある
- 1冊目候補: 「時のゆりかご世界案内」または「78枚カード解説書」

### コミュニティ
- 荒れない設計を最優先(ガイドライン先行、規模より安全)
