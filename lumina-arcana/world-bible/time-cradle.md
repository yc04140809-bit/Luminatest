- **ステータス**: ドラフト(最優先で設定を固める)

世界「時のゆりかご」と、その中心の聖域「生命の揺り籠」の詳細設定を積み上げる場所です(世界=時のゆりかご/聖域=生命の揺り籠。ユーザー決定 2026-07-18)。
公式定義・禁則は Master Library [02章](../master-library/02_time-cradle.md) が正であり、ここでは描写・構造・暮らしのディテールを育てます。

## 聖域「生命の揺り籠」

【正式設定】カードが静かに生まれ、光が集まり、生命と希望が育まれる神聖な場所。世界の中心にあり、Card Ceremony でカードが訪れる起点となる。

## 空間の構造【正式設定】(v1.0・2026-07-18)

世界は中心から外へ、静かな円環をなしている。

```
        月と星(天)
   ┌─────────────────┐
   │      森(蛍・風)      │
   │   ┌───────────┐   │
   │   │    湖(水面)    │   │
   │   │  ┌─────┐  │   │
   │   │  │ 世界樹  │  │   │
   │   │  │生命の揺り籠│  │   │
   │   │  └─────┘  │   │
   │   └───────────┘   │
   └─────────────────┘
```

- 中心: 世界樹。その枝葉が絡み合って**生命の揺り籠**を形づくり、抱いている
- 揺り籠と世界樹は静かな**湖**に抱かれ、水面は月と星の光を映す
- 湖を囲むのは深い**森**。蛍が灯り、風が葉を揺らす
- 天には**三日月**(受容と直感)と**一番星**(希望と導き)
- 世界の果て・その他の土地は【未設定】(必要になるまで定義しない)

## 時間の流れ【正式設定】(v1.0・2026-07-18)

- 現実より少しだけゆっくり流れる。心を整えるための時間
- 朝→昼→夕→夜、春→夏→秋→冬が巡る。巡りは循環と再生のしるし
- 過去は水面に映る景色のように、静かに眺められるもの

## 訪れる人々(v1.0・2026-07-18)

- 資格はいらない。疲れた心が「帰りたい」と思ったとき、誰でも帰ってこられる
- 世界は「おかえり。」と迎え、去るときは「いってらっしゃい。」と送り出す。引き留めることはない
- たどり着き方の物語上の描写は【未設定】(利用者体験としては「アプリを開く=帰ってくる」)
